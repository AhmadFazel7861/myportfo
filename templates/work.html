001 : Screenlight

Screenlight Blog is a dynamic blogging platform developed using Django. Key features include user authentication (signup, login, logout), post creation and management, threaded comments with reply functionality, post likes, view counts, bookmarking, blog search, author-wise blog filtering, and subscription to updates. Each blog displays an updated timestamp for version tracking, and the platform includes a custom admin dashboard for post moderation and user control.
