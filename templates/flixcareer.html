002 : FlixCareer

FlixCareer is a full-stack job application platform built with Django, HTML, CSS, and JavaScript. It allows users to create profiles, upload resumes, and apply for job listings. Job postings are managed exclusively through a secure admin dashboard—users cannot post jobs. Employers can review applications and shortlist candidates via the admin interface. The platform includes advanced job filtering (by location, type, and skills), a responsive dashboard for both user types, and JWT-based secure authentication.

"Scroll to land your dream job FlixCareer connects you to opportunity."
